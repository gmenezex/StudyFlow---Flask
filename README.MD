# 📊 StudyFlow - Sistema de Controle de Estudos + Dashboard

> Status do Projeto: 🚀 Em Desenvolvimento (Sprint 1)

O **StudyFlow** é uma aplicação web desenvolvida para ajudar estudantes a organizarem sua rotina acadêmica. Com ele, é possível gerenciar o tempo dedicado a cada disciplina, acompanhar o progresso através de um dashboard interativo e receber insights automáticos para otimizar o aprendizado.

---

## 🎯 Principais Funcionalidades

*   **🔑 Gestão de Usuários:** Cadastro de conta e login seguro com criptografia de senha.
*   **📚 Organização de Grade:** Cadastro e gerenciamento de matérias/disciplinas customizadas.
*   **⏱️ Cronômetro de Sessões:** Registro do tempo líquido dedicado aos estudos de forma prática.
*   **📈 Dashboard Estatístico:** Gráficos e métricas de desempenho para visualização da evolução do estudante.
*   **💡 Recomendações Inteligentes:** Sugestões simples de estudo baseadas nas matérias que precisam de mais atenção.

---
